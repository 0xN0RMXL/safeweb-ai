## GLOBAL PRESENTATION DESIGN SYSTEM

### Color Palette (MUST use these exact hex codes throughout)
```
BACKGROUNDS:
- Slide background:        #050607 (deep black-blue)
- Content panels:          #0A0C0E (slightly lighter black)
- Card backgrounds:        #12151A (dark card surface)
- Hover/highlight areas:   #1A1D23 (subtle light panel)

ACCENTS (Neon highlights):
- Primary neon green:      #00FF88 (main brand — headings, highlights, CTAs, key data, borders, glows)
- Green hover:             #00E67A
- Electric blue:           #3AA9FF (secondary — diagrams, subheadings, supporting accents)
- Blue hover:              #2E95E8

SEVERITY COLORS (for vulnerability/chart data):
- Critical:                #FF3B3B (bright red)
- High:                    #FF8A3D (orange)
- Medium:                  #FFD93D (yellow)
- Low/Safe:                #6BCF7F (soft green)
- Info:                    #3AA9FF (blue)

TEXT:
- Primary text (headings): #FFFFFF (pure white)
- Body text:               #B0B8C1 (light gray-blue)
- Muted/labels:            #6B7280 (gray)
- Disabled:                #4B5563 (dark gray)

BORDERS:
- Default borders:         #1F2937 (dark gray)
- Secondary borders:       #374151 (medium gray)
- Active/accent borders:   #00FF88 (neon green)
```

### Typography
```
- Headings (h1-h3):       Space Grotesk — Bold (700) / Semibold (600)
- Body text:               Inter — Regular (400) / Medium (500)
- Code / technical data:   JetBrains Mono or Fira Code — Regular (400)
- Terminal/hacker text:    JetBrains Mono — all monospaced elements
```

### Font Sizes
```
- Slide title (h1):        60-72px Space Grotesk Bold
- Section heading (h2):    36-48px Space Grotesk Bold
- Card heading (h3):       24px Space Grotesk Semibold
- Body text:               18-20px Inter Regular, color #B0B8C1
- Small labels/captions:   14px Inter Medium, color #6B7280
- Code snippets:           16px JetBrains Mono, color #00FF88
```

### Visual Effects & Decorations
```
- GREEN NEON GLOW:         Apply 0 0 15px rgba(0,255,136,0.3) glow behind key elements, titles, and accent borders
- BLUE NEON GLOW:          Apply 0 0 15px rgba(58,169,255,0.3) behind secondary elements
- GLASSMORPHISM CARDS:     #12151A at 80% opacity, backdrop blur 8px, 1px border #1F2937, rounded 12px
- GRADIENT ACCENTS:        Diagonal gradients from #00FF88 → #3AA9FF for hero highlights, dividers, progress bars
- MATRIX RAIN:             Faint green (#00FF88 at 4-6% opacity) falling code characters in background of select slides
- SCAN LINES:              Subtle horizontal scan line overlay at 3% opacity on dark backgrounds
- GRID PATTERN:            Faint dot-grid or line-grid pattern at 5% opacity in backgrounds
- GLITCH EFFECT:           Apply RGB-shift glitch displacement on the title slide logo/heading
- TERMINAL AESTHETIC:      Code blocks styled as terminal windows with green-on-black text, blinking cursor
- CIRCUIT BOARD TRACES:    Thin neon green (#00FF88 at 15% opacity) circuit-trace patterns as decorative borders
```

### Slide Layout Rules
```
- ALL slides: #050607 background, NO white/light backgrounds anywhere
- Content area: max-width 1100px, centered horizontally
- Padding: 80px from edges minimum
- One core idea per slide — NO information overload
- Left-heavy layouts for text+visual combos (60/40 split)
- Full-bleed visuals for screenshot/demo slides
- Consistent element positioning across all slides
- Every slide has a subtle top-right page number in #6B7280
```

### Transitions & Animations
```
- Between slides:          Smooth fade + subtle scale (1.02→1.0) transition, 400ms ease-out
- Title reveals:           Typewriter-style character-by-character reveal for heading text (green cursor)
- Content blocks:          Staggered fade-in from bottom with 80ms delay between items
- Chart/data reveals:      Bars/segments animate in sequentially with spring easing
- Code blocks:             Line-by-line typing animation simulating terminal output
- Diagrams:                Elements connect with animated neon-green trace lines
- Screenshots:             Slide in from right with perspective tilt, settling flat
- Glitch bursts:           Brief 120ms RGB-shift glitch on slide transitions (hacker aesthetic)
- Hover states:            Elements lift with translateY(-4px) + green glow intensifies
```
