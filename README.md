# SafeWeb AI - Web Application Vulnerability Scanner

A professional cybersecurity SaaS platform for scanning websites, files, and URLs for vulnerabilities and malware threats. Built with React, TypeScript, and Tailwind CSS.

## 🚀 Features

- **Comprehensive Vulnerability Scanning**: Website, file, and URL scanning
- **AI-Powered Analysis**: Advanced threat detection and remediation guidance
- **Detailed Reports**: Export scan results in multiple formats
- **Educational Resources**: Learn about web security best practices
- **Real-time Monitoring**: Track scan progress and history
- **Professional Dashboard**: Intuitive interface for managing security scans

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Routing**: React Router v6
- **Code Quality**: ESLint + TypeScript

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
